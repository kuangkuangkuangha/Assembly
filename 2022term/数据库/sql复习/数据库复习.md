##### 5.2 参照完整性

- 定义参照完整性

  ```sql
  create table SC
  (
  			Sno char(9)not null,
  			Cno char(4)not null,
  			grade smallint,
  			primary key(sno, cno),
  			foreign key(sno) referrences Student(sno),   // 定义参照完整性
  			foreign key(cno) referrences Course(cno)
  )
  ```

  

- 参照完整性检查和违约处理

  - 拒绝（no action）执行
